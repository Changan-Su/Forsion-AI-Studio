# v2026.03.28 - Apple-Style UI & Animation Polish

## 概述

引入 Framer Motion 动画库，对 Forsion AI Studio 全局 UI 动画和交互反馈进行 Apple 风格优化。解决了之前组件大量使用条件渲染（`&&` / ternary）导致的「弹出即消失、无过渡」问题，全面提升了视觉流畅度。

## 改动内容

### 1. 新增 framer-motion 依赖
- `client/package.json` 新增 `framer-motion` 作为 runtime 依赖

### 2. 新建 AnimatedUI.tsx 动画基础组件库
- 路径：`client/components/AnimatedUI.tsx`
- 包含 7 个可复用的动画组件：
  - `AnimatedModalBackdrop` — 背景遮罩淡入/淡出
  - `AnimatedModalContent` — 弹窗内容缩放+透明度弹簧动画（入场 scale 0.96→1，出场 scale 0.97→0）
  - `AnimatedPanel` — 从左/右侧滑入的面板（弹簧物理）
  - `AnimatedOverlay` — 简单淡入淡出遮罩（用于移动端侧栏）
  - `AnimatedDropdown` — 下拉菜单缩放+透明度（可选 transformOrigin 方向）
  - `AnimatedListItem` — 列表项进场动画（淡入+上滑，支持 stagger）
  - `AnimatedCollapse` — 展开/折叠高度过渡动画
- 定义了 3 套 Apple 风格弹簧配置：`appleSpring`, `gentleSpring`, `snappySpring`

### 3. 弹窗/模态框动画（最大视觉提升）
- **SettingsModal** — 背景淡入，内容弹簧缩放入/出场
- **AgentConfigPanel** — 同上
- **AuthorizationDialog** — 同上
- **RegisterModal** — 同上
- **Expanded Composer** — 移除了旧的 `@keyframes modalBounce` CSS 动画，改用 Framer Motion 弹簧动画，入场和出场均有流畅过渡
- **FileSizeError 对话框** — 同上
- 所有弹窗在 `App.tsx` 中用 `<AnimatePresence>` 包裹，实现出场动画

### 4. 面板滑入动画
- **WorkspacePanel** — 从右侧滑入/滑出（弹簧物理），替代了原来的 `if (!isOpen) return null` 即开即关
- **Sidebar 移动端遮罩** — 背景淡入/淡出，替代了原来的瞬间出现

### 5. 下拉菜单/弹出层动画
- **Model Dropdown**（App.tsx）— scale+opacity 从 top-left 方向缩放入场
- **CommandAutocomplete** — 从 bottom 方向缩放入场
- **EmojiPicker** — 轻微缩放+淡入
- **Sidebar 右键菜单** — scale+opacity 从 top-right 方向

### 6. 聊天消息动画
- **消息入场** — 新消息使用 `motion.div` 从下方滑入+淡入（仅动画最新消息，已有消息不重新动画）
- **Processing 指示器** — 入场/退场动画 + 弹跳的三点加载指示
- **空状态页** — 使用 Framer Motion 替代旧的 CSS `animate-fade-in-up`

### 7. 折叠/展开动画
- **ToolCallBlock** — 展开/折叠使用 `AnimatedCollapse`（平滑高度过渡）
- **ThinkingBlock** — 思考过程的展开/折叠使用 `AnimatedCollapse` + chevron 旋转动画
- **Sidebar 归档区域** — 展开/折叠使用 `AnimatedCollapse` + chevron 旋转动画

### 8. 微交互优化
- 按钮按下反馈：`active:scale-95` 改为更精细的 `active:scale-[0.97]`
- 开关控件：knob 添加 `cubic-bezier(0.4, 0, 0.2, 1)` 弹簧曲线
- hover 过渡统一为 `duration-200`
- 滚动条：宽度 8px→6px，颜色改为半透明，Firefox 兼容 `scrollbar-width: thin`

### 9. 视觉润色
- 所有 glass 类（`.glass`, `.glass-dark`, `.apple-glass`, `.forsion1-glass`）添加 `inset 0 1px 0 rgba(...)` 内光效果，增加玻璃质感深度
- Dark mode 下的 glass 效果也添加了微妙的内光
- 遮罩背景颜色统一从 `bg-black/60` 调整为 `bg-black/50`，更通透

## 技术要点
- 所有动画使用 GPU 加速属性（`transform` + `opacity`）
- `AnimatePresence` 解决了 React 条件渲染无法做退出动画的核心问题
- 所有改动都是**叠加式**的，不破坏现有 Tailwind 类和主题系统
- 构建验证通过（`vite build` 成功）
