## 布局优化

### 2. 边框圆角 Token 系统

**文件：** `client/index.html`

在 `:root` 中新增 6 个 CSS 自定义属性：

```css
--radius-xs: 8px;    /* 行内代码、标签内角 */
--radius-sm: 12px;   /* 按钮、下拉菜单项 */
--radius-md: 16px;   /* 卡片、发送按钮 */
--radius-lg: 22px;   /* 消息气泡、面板 */
--radius-xl: 28px;   /* 输入框、模态框 */
--radius-pill: 9999px; /* 头像、开关、徽章 */
```

同时在 Tailwind config `extend.borderRadius` 中以 `token-*` 前缀注册，便于 JIT 类名使用（`rounded-token-xl` 等）。

这解决了之前 `rounded-lg`、`rounded-2xl`、`rounded-[2rem]`、`rounded-[26px]`、`rounded-[32px]` 混用导致的视觉不协调。

### 3. 侧栏与 Header 比例调整

**文件：** `client/components/Sidebar.tsx`、`client/App.tsx`

| 元素 | 之前 | 之后 |
|------|------|------|
| 侧栏宽度（移动端固定抽屉） | `w-72` (288px) | `w-64` (256px) |
| 侧栏宽度（桌面端） | `md:w-64` (256px) | `md:w-56` (224px) |
| Header 高度 | `h-16` (64px) | `h-[52px]` (52px) |

视觉权重从 sidebar/header 转移到核心对话区。

### 4. 消息宽度收窄

**文件：** `client/components/ChatArea.tsx`、`client/App.tsx`

- 消息行：`max-w-4xl` → `max-w-3xl`
- 输入框表单：`max-w-4xl` → `max-w-3xl`

对于较宽屏幕，文字行不再无限拉伸，阅读体验更接近 Apple Messages / iMessage 的紧凑风格。

## 附件预览重设计

**文件：** `client/App.tsx`（composer 区域），`client/components/ChatArea.tsx`（消息内联附件）

### 之前的问题

- 附件以 `h-20` 的独立方块悬浮在输入框**外部上方**，与输入框在视觉上断开
- 删除按钮以 `absolute -top-2 -right-2` 红色圆形溢出，视觉粗糙
- 没有入场动画

### 新设计：Chip 内嵌输入框

**Composer 附件：**
- 移除旧的 `mb-3 ml-2` 外部容器，将附件区域**移入**输入框容器内部
- 输入框容器从 `flex items-end` 改为 `flex flex-col`，顶部为 chip 行，下方为输入行
- 每个 chip：40×40 圆角缩略图（图片）或图标（文档）+ 文件名 + 类型 + hover 显示的删除按钮
- chip 入场动画：framer-motion spring（stiffness 400, damping 28），支持 `AnimatePresence` exit
- 删除按钮默认 `opacity-0`，`group-hover/chip:opacity-100`，hover 变红

**消息内联附件（ChatArea）：**
- 图片：改为 `rounded-[var(--radius-md)]`，底部加文件名覆盖层（`bg-black/40 backdrop-blur-sm`）
- 文档：改为 chip 风格（40×40 图标 + 文件名 + 类型）

## 文字框原地展开（替代模态窗口）

**文件：** `client/App.tsx`

### 移除
- `expandedInput` state 变量及所有同步逻辑
- 整个 `AnimatedModalBackdrop` + `AnimatedModalContent` 展开输入模态（约 90 行）
- "Apply" / "Cancel" 按钮及 `setInput(expandedInput)` 同步

### 新增机制
- 保留 `isInputExpanded` boolean state
- 输入框容器通过内联 `style` 做 `border-radius` 过渡（300ms `cubic-bezier(0.25,0.1,0.25,1)`）：展开时 `var(--radius-xl)` → `var(--radius-lg)`
- Textarea `style.maxHeight` 动态切换：折叠时 `200px`，展开时 `50vh`
- Textarea 追加 `transition-[max-height] duration-300` class
- 展开后底部显示 footer row：`Ctrl+Enter to send · Esc to collapse` + Collapse 按钮
- 展开按钮图标旋转 45deg（内联 `style.transform`，300ms 过渡）
- 键盘支持：
  - `Escape` → 收起展开（同时关闭命令自动补全）
  - `Ctrl+Enter`（展开状态）→ 直接发送并收起
