# Function Review: Monet 2.0 主题（清透水彩莫奈风）

**版本**：v1.0
**日期**：2026-03-26
**关联 PRD**：[prd_v1.0.html §四 用户体验 - 多主题风格](../prd/prd_v1.0.html)

---

## 功能描述

Monet 2.0 是 Forsion AI Studio 的清透水彩风主题，替代了原版 Monet Cliffs。原版问题是背景梯度偏灰暗（`#a8c5d0 → #d4b5c8 → #5a5c8a`，尾段深紫蓝），加上高模糊毛玻璃（`blur(40px)`），整体观感灰蒙蒙、不够清澈。Monet 2.0 通过提亮背景梯度、减少模糊量、增加饱和度、换用更干净的强调色，实现"晴天莫奈睡莲池"般的通透清澈感。

## 实现方案

### 技术选型

纯 CSS 变量 + `backdrop-filter` 毛玻璃。通过 `data-theme="monet"` 属性选择器切换整套 CSS 变量和组件样式，零 JS 依赖。

### 核心实现

#### 1. CSS 变量定义

```css
[data-theme="monet"] {
  --bg: rgba(255,255,255,.4);
  --bg-card: rgba(255,255,255,.6);
  --bg-hover: rgba(255,255,255,.72);
  --border: rgba(255,255,255,.5);
  --text: #2c3e6b;           /* 深海军蓝，对比度高 */
  --text2: #4a5d88;
  --text3: rgba(44,62,107,.45);
  --accent: #3d68b5;         /* 清澈蓝，替代旧版灰紫 #3e406f */
  --accent-bg: rgba(61,104,181,.07);
  --accent-border: rgba(61,104,181,.16);
  --success: #2aab6e;
  --warn: #e49830;
  --error: #e04e4e;
  --glass-bg: rgba(255,255,255,.75);
  --glass-border: rgba(255,255,255,.55);
  font-family: 'Inter', sans-serif;
  background: linear-gradient(135deg, #b8e6f2 0%, #f2d4e2 42%, #a8c4eb 100%);
  color: var(--text);
}
```

**与旧版 Monet Cliffs 的关键区别：**

| 维度 | Monet Cliffs（旧） | Monet 2.0（新） | 改进原因 |
|------|-------------------|----------------|---------|
| 背景梯度 | `#a8c5d0 → #d4b5c8 → #5a5c8a` | `#b8e6f2 → #f2d4e2 → #a8c4eb` | 全段提亮去灰，尾段从深紫蓝改为清透蓝 |
| 毛玻璃 | `blur(40px) saturate(160%)` | `blur(24px) saturate(200%)` | 减少模糊保清晰，提高饱和度让色彩更通透 |
| 卡片透明度 | 65% 白 | 75% 白 | 更清晰的卡片边界 |
| 文字色 | `#4a4b6a`（灰紫调） | `#2c3e6b`（深海军蓝） | 更高对比度，更干净 |
| 强调色 | `#3e406f`（暗紫灰） | `#3d68b5`（清澈蓝） | 更有活力，去灰 |
| Skill 图标 | 纯色 `#4a4b6a` | 每个独立渐变色 | 更丰富的视觉层次 |
| 标题字体 | `Dancing Script` 手写体 | `Dancing Script` 手写体 | 保留印象派气质 |

#### 2. 组件样式

```css
/* 毛玻璃卡片 */
[data-theme="monet"] .card-glass {
  background: var(--glass-bg);
  border: 1px solid var(--glass-border);
  backdrop-filter: blur(24px) saturate(200%);
  box-shadow: 0 2px 16px -4px rgba(44,62,107,.06),
              0 0 0 .5px rgba(255,255,255,.6);
}

/* 用户气泡 */
[data-theme="monet"] .user-bubble {
  background: linear-gradient(135deg, #3d68b5, #5a8ad5);
  color: #fff;
  border: 1px solid rgba(255,255,255,.2);
}

/* Skill 图标（每个独立渐变色） */
[data-theme="monet"] .skill-icon-1 { background: linear-gradient(135deg, #4a90d9, #68b8e8); }
[data-theme="monet"] .skill-icon-2 { background: linear-gradient(135deg, #52b88a, #7ad4a8); }
[data-theme="monet"] .skill-icon-3 { background: linear-gradient(135deg, #e8a060, #f0c088); }
[data-theme="monet"] .skill-icon-4 { background: linear-gradient(135deg, #9078c0, #b098d8); }
[data-theme="monet"] .skill-icon-5 { background: linear-gradient(135deg, #5098c8, #78b8dd); }

/* 主按钮 */
[data-theme="monet"] .btn-primary {
  background: rgba(61,104,181,.88);
  color: #fff;
  backdrop-filter: blur(6px);
}

/* 导航高亮 */
[data-theme="monet"] .nav-pill.active {
  background: rgba(255,255,255,.5);
  color: #2c3e6b;
  backdrop-filter: blur(6px);
}

/* 模块标题 */
[data-theme="monet"] .module-title {
  font-family: 'Dancing Script', cursive;
  font-size: 1.5rem;
  color: #2c3e6b;
}

/* 遮罩背景 */
[data-theme="monet"] .overlay-backdrop {
  background: rgba(61,104,181,.15);
  backdrop-filter: blur(12px) saturate(160%);
}
```

## 接口与适配

本主题设计可移植到任何使用 CSS 变量的前端项目。适配步骤：

1. **引入字体**：加载 `Inter` 和 `Dancing Script`（Google Fonts）
2. **复制 CSS 变量块**：将 `[data-theme="monet"]` 及其组件样式复制到目标项目
3. **匹配 CSS 变量名称**：目标项目需使用相同的 CSS 变量名（`--bg`, `--bg-card`, `--text`, `--accent` 等），或建立映射
4. **激活主题**：在根容器上设置 `data-theme="monet"`

**设计要点复用清单：**
- 背景梯度三色值：`#b8e6f2`, `#f2d4e2`, `#a8c4eb`
- 毛玻璃参数：`blur(24px) saturate(200%)`
- 文字色阶梯：`#2c3e6b` → `#4a5d88` → `rgba(44,62,107,.45)`
- 强调色：`#3d68b5`
- 卡片白背景：`rgba(255,255,255,.75)`
