Forsion AI Studio — Layout Redesign

以下是三个核心改进的交互 Demo：布局优化、附件区域重新设计、以及文字框原地展开。 点击 Demo 中的按钮可以体验交互效果。

改进 1：布局 — 侧栏比例收窄（240→224px），header 高度降低（64→52px）， 消息区域留白更均匀，input 区域视觉权重降低但功能不变。
改进 2：附件预览 — 附件以「chip」形式内嵌在输入框内部（不是外部独立行）， 图片有圆角缩略图+文件名，文档有图标+文件名。入场有弹簧动画，hover 显示删除按钮。
改进 3：文字展开 — 点击展开按钮后，输入框**原地**平滑拉高到 60vh， 而不是弹出一个新的模态窗口。收回时平滑缩小回原始高度。圆角随高度平滑过渡。
改进 4：圆角一致性 — 定义 6 级圆角 token（8/12/16/22/28/pill）， 所有 UI 元素对齐到这些 token，避免 rounded-lg 和 rounded-2xl 混用的视觉错位。

好的，我来分析这张设计稿。主要有以下几个问题：

1. 视觉层级不清晰 — header 和侧栏的权重过重，挤压了核心对话区的空间。

2. 圆角不统一 — 混合使用了 8px、12px、16px、24px、32px 等多种圆角，缺乏系统性。

3. 过渡生硬 — 大部分 UI 变化没有动画过渡，造成视觉跳跃。

Border Radius Token System

定义 6 个圆角级别，所有 UI 元素严格对齐到这些 token，避免混用导致的视觉不协调。

Content

XS — 8px

Inline code, small tags, inner corners of messages

Content

Buttons, session items, tooltips, dropdowns

Content

MD — 16px

Cards, attachment chips, code blocks, send button

Content

LG — 22px

Message bubbles, popovers, panels

Content

XL — 28px

Input composer, modals, settings panel

Content

Avatar, toggle switch, badge, icon-only buttons
