## 版本目标

将 Forsion AI Studio 从实验性 v0.4.x 升级为正式 v1.0 MVP，核心目标：
1. 稳定现有多模型对话功能
2. 完善 Skills 体系（Skills 市场、全局/Session 启用、官方 Skills 管理）
3. 建立跨应用统一数据中枢（用户级数据分区 + App ID 存储）

## 版本变更摘要

从 v0.4.4（实验性迭代）跃升至 v1.0（正式 MVP），重新定义产品定位为"跨应用统一个人 AI Agent 平台"，新增 Skills 市场和跨应用数据中枢能力规划。

PRD v1.0 最终版包含：
- §五 详细方案：10 个功能模块，每个含 Mermaid 流程图 + 业务规则 + 原型链接 + AC 验收标准
- §十一 开发任务清单：8 个 Epic / 35 个 Task（P0: 17, P1: 18），每个 Task 关联 AC
