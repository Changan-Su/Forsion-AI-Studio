## 版本目标

将 Forsion AI Studio 从实验性 v0.4.x 升级为正式 v1.0 MVP，核心目标：
1. 稳定现有多模型对话功能

3. 建立跨应用统一数据中枢（用户级数据分区 + App ID 存储）
