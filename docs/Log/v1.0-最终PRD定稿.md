### 2. §十一 开发任务清单（新增）

将功能拆解为可执行的 Task，供 forsion-dev 逐条实现：

- **8 个 Epic**：Skills 市场与作用域、Skill 执行可视化、跨应用数据中枢、多主题风格、对话历史云同步、新用户引导、官方 Skills 管理、用户画像与数据同步
